🚀 3가지 방식 성능 비교 분석

기존 방식 vs 뷰 방식 vs 상위 100개 뷰 방식

어떤 방식이 최적일까요?

🟢 기존 방식

특징: 인덱스 + 저장된 like_count 컬럼 사용

장점: 빠른 응답속도, 일정한 성능

단점: 좋아요 변경 시 업데이트 필요

🔴 뷰 방식

특징: 실시간 계산 (GROUP BY + COUNT)

장점: 100% 정확한 데이터

단점: 느린 응답속도, 데이터 증가시 성능 저하

🟠 상위 100개 뷰

특징: 미리 계산된 상위 100개만 저장

장점: 뷰 방식보다 빠름

단점: 여전히 뷰 방식의 한계 존재
